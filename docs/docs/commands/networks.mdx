# Networks

Running `sedge networks` will give you a list of supported networks

## Help

```
$ sedge networks
List supported networks

Usage:
  sedge networks [flags]

Flags:
  -h, --help   help for networks

Global Flags:
      --log-level string   Set Log Level, e.g panic, fatal, error, warn, warning, info, debug, trace (default "info")
```
