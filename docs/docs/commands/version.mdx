# Version

Running `sedge version` will give you the version you are running while using sedge.

## Help

```
$ sedge version -h
Initializing configuration
Print sedge version

Usage:
  sedge version [flags]

Flags:
  -h, --help   help for version

Global Flags:
      --log-level string   Set Log Level, e.g panic, fatal, error, warn, warning, info, debug, trace (default "info")
```

## Execution Example

```
$ sedge version
2022-00-00 00:00:00 -- [INFO] [Logger Init] Log level: info
2022-00-00 00:00:00 -- [INFO] You are running the latest version of sedge. Version:  v1.8.0
sedge v0.5.1
```
